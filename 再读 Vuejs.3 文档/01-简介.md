# 简介

Vue.js 一款渐进式构建用户界面的 JavaScript 框架，具有易学易用、性能出色、使用场景丰富等特点，还提供了一套声明式、组件化的编程模型，可以高效的开发用户界面。

## 渐进式框架：

在多样化的 Web 开发中，不同的开发者在构建 Web 应用时的形式和规模会有很大的不同，基于此，Vue 框架非常注重灵活性，可以根据自己的需求场景以不同的方式使用 Vue：

* 无需构建步骤，进行时增强静态的 HTML；
* 在任何页面作为 Web Components 嵌入；
* 单页面应用（SPA）；
* 全栈或服务端渲染（SSR）；
* Jamstack 或静态站点生成（SSG）；
* 桌面端、移动端、WebGL、甚至命令行终端的界面；

## 单文件组件：

在启用构建工具的 Vue 项目中，可以使用单文件组件（英文 Single-File Components，缩写 SFC）这一 Vue 标志性功能将模板（HTML）、逻辑（JavaScript） 和 样式（CSS） 封装到同一个 `*.vue` 文件。

## API 风格：

Vue 支持通过选项式 API 和组合式 API 两种风格编写组件。
